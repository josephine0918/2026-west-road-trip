# US West Road Trip Website

## Update on GitHub Pages
1. Create a GitHub account if you do not have one.
2. Create a new repository, for example: `us-west-roadtrip`.
3. Unzip this file.
4. Upload all files in the unzipped folder to the repository root:
   - index.html
   - styles.css
   - data.js
   - script.js
   - README_netlify_update.txt
5. Go to Repository Settings → Pages.
6. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
7. Save.
8. Wait 1–3 minutes. Your site will be available at:
   https://YOUR-GITHUB-USERNAME.github.io/us-west-roadtrip/

If you edit files later, upload the new files to the same repository. GitHub Pages will update automatically.
